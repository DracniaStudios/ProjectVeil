// Core Functions
1. Player can play in both 2D and 3D modes, with seamless switching between the two.

2. Player can move in all directions (up, down, left, right) in 2D mode, and can look around and move in 3D mode.

3. Render 2D function Of the game to understand how to draw the game in 2D mode.

4. Render 3D function Of the game to understand how to draw the game in 3D mode.

// Order Of Operations:

1. Draw Camera

1.1: Scenes Will Have 2 Modes: 2D and 3D on drawMethod and 1 Update method that will update the game state and call the appropriate draw method based on the current mode.
1.2: Render Player Sphere in 3D space
1.3: Render Player Rectangle in 2D space
1.4: Clamp Player3D to map size (1920x1080)
1.5: Clamp Player2D to map size (1920x1080)

2. Draw Map

3. Draw Player

4. Resolve Collisions

5. Move Player
